// MÉTODO DE NEWTON PARA CALCULAR A RAIZ QUADRADA DE UM NÚMERO //


INÍCIO

	DEFINIR VARIÁVEL N.

	EXIBA "QUAL O NÚMERO DO QUAL DESEJA OBTER A RAIZ QUADRADA? ".

	N RECEBE O VALOR DIGITADO PELO USUÁRIO.

	DEFINIR VARIÁVEL B = 2.

	DEFINIR VARIÁVEL P = (B + (N/B) / 2).

	P = P ** 2.

	FAÇA:

	SE A DIFERENÇA ABSOLUTA (N - P) < 0,0001:

		EXIBA ("(N) AO QUADRADO EQUIVALE À (P)").

		FINALIZAR OPERAÇÃO.

	FIM SE 

	B = P

	P = (B + (N / B) / 2).

	P = P ** 2.
